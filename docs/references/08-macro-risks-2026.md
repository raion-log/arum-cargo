## 1. 거시경제 환경 (2026)

### 1.1 한국 경제
- **한국은행 전망**: **2.0% 저성장** 기조
- **환율**: **1,400원대 중반 고환율** 고착화
- **소비자 심리**: 위축

### 1.2 글로벌 공급망
- **KOTRA 전망**: 글로벌 공급망 재편 + **관세 조치로 인한 무역 불확실성** 증대
- **미국 관세**: 15% 수준 관세 조치 → 수출입 물동량 영향
- **공급망 다변화 필요성** ↑

### 1.3 산업 흐름
- **글로벌 물류 DX**: +45% 가속
- **IT·반도체 화물 수요**: +35% (고부가가치 유지)
- **AI 투자 확대**: +60% (데이터센터·피지컬 AI)
- **2.0% 성장 모멘텀**: 한국 완만한 성장 + 외환 변동성

## 2. RAION의 노출 리스크 (What could go wrong)

### 2.1 거시적 리스크

| 리스크 | 영향 | 확률 |
|---|---|---|
| 환율 급등으로 유저 심리 위축 → 교육·이미지 메이킹 지출 감소 | A-Side Affiliate 수익 감소 | 중 |
| 관세 인상으로 물동량 감소 | L-Side 확장 지연 | 중~고 |
| 국내 저성장 → 항공업계 투자 위축 → 채용 공고 감소 | A-Side 콘텐츠 풀 축소 | 중 |
| API 공급자(항공사) 요금 인상 | 원가 압박 | 저~중 |

### 2.2 운영적 리스크

| 리스크 | 영향 | 완화 방안 |
|---|---|---|
| 네이버 뉴스 API 쿼터 초과 | 뉴스 카드 공백 | 캐싱 + 다중 소스 (RSS) 이중화 |
| AviationStack 무료 티어 소진 | 운항 정보 미표시 | 캐싱 + OpenSky 폴백 |
| 뉴스 저작권 이슈 | 서비스 중단 리스크 | 제목+요약+원문링크만, 출처 명시 |
| 이메일 스팸 필터링 | 전달률 저하 | 도메인 인증 + 더블 옵트인 |

## 3. 대응 전략 — "High-Value Focus"

블루프린트의 핵심 대응 원칙: **거시 위기는 고부가가치로 상쇄한다.**

### 3.1 고부가가치(IT·반도체) 화물 네트워크 집중 공략
- **환율 상승 시 반도체 등 수출 주도 기업 이익은 오히려 극대화** → 이들이 플랫폼의 주요 화주 풀이 됨
- 전 세계 AI 인프라 투자 지속으로 **반도체 등 Physical AI 관련 항공 수출 물동량 견조**
- 플랫폼은 **단가 방어가 용이한 IT·반도체 화물 네트워크 포워더 확보에 집중**
- ⏸ Phase 2 L-Side에서 본격 실행

### 3.2 고성장 세그먼트 포트폴리오 다각화
- 일반 화물 감소 대비: **이커머스, 의약품(콜드체인), 항공우주 부품 등 시간 민감성 특수 화물(Special Cargo)** 로 확장
- ⏸ Phase 3 계획

### 3.3 AI 기반 적재 공간 최적화를 통한 비용 절감 가치 제공
- 디지털 기획 툴을 통해 서류상 부피 불일치 바로잡고 화물 적재 최적화
- **가용 공간 약 23.6% 추가 확보** 가능 (블루프린트 인용)
- 고환율 속 화주·포워더에게 확실한 비용 절감 가치 제공 → **락인 효과 극대화**
- ⏸ Phase 2 L-Side 구현

## 4. MVP 단계 리스크 대응 (즉시 적용)

Phase 1 MVP에서 위 거시 전략을 직접 구현하기는 어렵지만, **다음은 즉시 반영해야 한다**:

### 4.1 저비용·무료 API 우선
- MVP는 **유료 API 구독 없이 런칭** 가능하도록 설계
- 네이버 뉴스 + AviationStack 무료 + OpenSky + 워크넷 공공 API + RSS
- 유료 전환은 실제 트래픽 검증 후

### 4.2 운영비 최소화
- Vercel Hobby 플랜 (무료)
- Supabase Free Tier (500MB DB, 50K MAU)
- Resend Free Tier (3,000통/월)
- **MVP 3개월간 운영비 = 0원** 목표

### 4.3 콘텐츠 자동화 우선
- 뉴스·채용 공고 자동 수집 → 수동 큐레이션 시간 최소화
- 에디터 인건비 = 0 (초기 라욘 본인 직접 운영)

### 4.4 크로스 시너지로 LTV ↑
- A-Side에서 모은 이메일 구독자 → I-Side 뉴스 다이제스트 → 재방문 → 광고·제휴 노출
- **1명 획득 비용(CAC) = 거의 0**, 운영비 절감
