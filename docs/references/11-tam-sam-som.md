# TAM / SAM / SOM — 항공 화물 시장 규모 및 Segment Map

> **버전**: v0.3 (Cargo-First Pivot)
> **근거**: [ADR-008](../adr/ADR-008-pivot-to-cargo-first.md) — 아름 카고 (Arum Cargo) 전략 피벗
> **목적**: 아름 카고가 겨냥하는 **한국 항공 화물 생태계** 시장을 3단계로 좁혀 정의하고, PRD 작성 전에 "누구를 얼마나 실제로 잡을 수 있는가"를 명시한다.
> **주의**: 아래 수치는 공개 통계 + 업계 현직자(사용자 본인 11년차) 경험치 기반 **추정치**이다. 검증 필요 항목은 [../open-questions.md](../open-questions.md)의 OQ-R18~R20에 등록한다.

---

## 0. 요약 (한 줄)

- **TAM**: 한국 항공 화물 생태계 전체 종사자 **약 21,000명** (항공사 화물 + 콘솔사 + 포워더 + 지상조업 + 관세사 + 학과 재학·졸업)
- **SAM**: 아름 카고 MVP(뉴스·채용·이메일)가 디지털로 도달 가능한 **약 15,700명** (75% 디지털 침투율)
- **SOM**: Phase 1~5 MVP 12개월 내 **WAU 500명** (SAM 3%), Phase 6~7 장기 **WAU 8,000명** (SAM 51%)
- **C1 핀포인트** (2~5년차 콘솔사·포워더 영업·오퍼): 약 **4,000명** — 모든 PRD 결정의 기준점

## 1. TAM — Total Addressable Market

### 정의
**"대한민국 항공 화물 생태계의 모든 종사자와 인접 이해관계자"** — 아름 카고가 장기적으로(5년+) 가치를 제공할 수 있는 최대 경계.

### 구성 (종사자 수 기준)

| 하위 세그먼트 | 추정 규모 (명) | 비고 |
|---|---|---|
| **항공사 화물 부서** (대한항공카고·아시아나카고·LCC 화물팀) | 약 3,000 | 영업·오퍼·예약·서비스·Ops. 대한항공카고가 단일 최대 고용처 |
| **콘솔사** (대형 콘솔레이터, 코스모항운·우정항공·트리플크라운인터내셔널·서울항공 등) | 약 2,500 | 국제 항공화물 콘솔 업무. 2~10년차가 가장 두꺼운 층 |
| **국제물류·포워더 항공 담당** (판토스·CJ대한통운·한진·세방 등 종합 포워더의 항공과) | 약 10,000 | 해상 겸업 다수. 순수 항공 담당자만 추정 |
| **공항 상주 지상조업·화물터미널** (스위스포트·KAS·인천공항터미널 직원 중 화물) | 약 2,000 | ULD 핸들링, 트럭 도킹, 화물 검수 |
| **관세사·통관사** (항공 통관 전담) | 약 1,000 | 항공 전담 관세사무소 직원 |
| **항공 물류 관련 학과** (재학생 + 최근 2년 졸업자) | 약 2,500 | 한서대·인하대·한국항공대·중원대·청주대·가톨릭관동대 합계 추정 |
| **총 TAM** | **약 21,000명** | 중복 제거 추정 |

### 참고: 매출 기준 TAM
- 국내 항공화물 산업 연간 매출 **약 8~10조원** (한국무역협회 / IATA Air Cargo Market Analysis 2024 기준)
- 글로벌 항공화물 시장 규모 **약 150조원** (IATA 2024)
- 아름 카고의 비즈니스 모델은 **종사자 네트워크 중심**이므로 매출이 아닌 **종사자 수**가 실질 TAM 기준.

### 출처·검증 상태
- 한국무역협회 항공물류 통계 (2024)
- 국토교통부 항공정책실 연간 통계
- 사용자 본인 11년차 업계 경험치 (콘솔사·포워더 직원 수 체감)
- ⚠️ **검증 필요**: 포워더 항공 담당자 정확 수치 → [../open-questions.md](../open-questions.md) OQ-R18

### TAM 의미
- TAM은 아름 카고의 **"천장"**. 장기 비전("아름" 브랜드 전체 항공 생태계)으로 확장해도 여전히 종사자 기준으로는 제한적임.
- 좁은 TAM은 오히려 **Moat**가 된다: 대형 플랫폼(사람인·잡코리아)이 비용 대비 진입할 유인이 없는 버티컬.
- MVP 단계에서 TAM 전체를 겨냥하지 않는다. 대신 SAM/SOM으로 좁혀 들어감.

## 2. SAM — Serviceable Available Market

### 정의
**"아름 카고의 현재 프로덕트 구조(웹 기반, 이메일 다이제스트, 채용 큐레이션)로 합리적 마케팅 내 도달 가능한 시장"** — 법적·기술적·언어적 장벽 없이 서비스를 붙일 수 있는 영역.

### 필터 조건
1. **디지털 접근 가능** (스마트폰 + 이메일 사용) — 2026년 기준 화물 업계 디지털 침투율 약 **75%** (시니어층 25% 제외)
2. **한국어 중심** (영어 원문 용어 병기)
3. **무료 가치** (유료 전환 없음 — ADR-008 수익화 제거 결정 반영)
4. **MVP 스코프 = 뉴스 다이제스트 + 채용 큐레이션 + 이메일 구독**

### SAM 세그먼트 (페르소나 매핑)

| 세그먼트 | 페르소나 코드 | 추정 규모 (명) | 디지털 침투율 | SAM |
|---|---|---|---|---|
| **2~5년차 콘솔사·포워더 영업·오퍼** (★ 핀포인트) | **C1** | 약 5,000 | 80% | **약 4,000** |
| **1년차 신입 (화물 입사 직후)** | C2 | 약 1,500 | 90% | 약 1,350 |
| **6~10년차 시니어 (팀장급)** | C3 | 약 3,500 | 70% | 약 2,450 |
| **항공교통물류학과 재학·졸업예정자** | A1 | 약 2,500 | 95% | 약 2,375 |
| **콘솔사·포워더 채용 담당자** (HR) | B1 | 약 300 | 85% | 약 255 |
| **항공사 화물 부서 2~10년차** (대한항공·아시아나 카고 등) | (인접) | 약 2,000 | 75% | 약 1,500 |
| **종합 포워더 항공과 주니어** (판토스·CJ 등) | (인접) | 약 5,000 | 75% | 약 3,750 |
| **합계 (중복 제거 추정)** | — | — | — | **약 15,700** |

### SAM 의미
- 아름 카고 MVP 기능으로 **"이론상 유입 가능한 최대 유저"**.
- SAM 규모는 크지 않지만(약 1.5만명), **도달 가능한 전원이 고수요·고재방문 유저**라는 것이 특징.
- C1 4,000명 + C3 2,450명 = **핵심 활성층 약 6,450명**. 이들이 WAU의 중추.

### ⚠️ 검증 필요 (OQ-R19)
- 콘솔사·포워더 직원 연령·연차 분포 정확 수치 (현재 사용자 경험치 기반)
- 디지털 침투율 75% 가정의 근거 강화 — LinkedIn 한국 화물 그룹 규모 간접 측정

## 3. SOM — Serviceable Obtainable Market

### 정의
**"Phase별 현실적으로 확보 가능한 WAU(Weekly Active Subscribers)"** — 1인 운영·제로 마케팅 예산·익명 운영(11년차 도메인 전문성만 보유) 가정.

### 산정 논리
1. **Phase 1~4 (PRD~API 연동)**: 유저 제로, 시드 준비 기간
2. **Phase 5 MVP 런칭 후 1~3개월**: 네이버 카페("항공화물종사자") + LinkedIn 한국 화물 그룹 + 지인 직군 확산
3. **Phase 5 3~6개월**: 에디터 Pick 품질 입소문 + 이메일 다이제스트 공유 루프
4. **Phase 5.5 (6~12개월)**: `/employers` 공개 → B1 유입 → 채용 공고 품질 향상 → C1 재방문 강화 (양면 시장 플라이휠 점화)
5. **Phase 6~7**: 커뮤니티(Remember 벤치마크) + 모바일 앱 → SAM 대부분 흡수

### SOM 목표 (Phase별 WAU)

| Phase | 기간 | WAU 목표 | SAM 대비 | 전제 |
|---|---|---|---|---|
| **Phase 5 (MVP)** | 0~6개월 | **500** | 3% | 네이버 카페·LinkedIn 초기 유입, 지인 직군 확산 |
| **Phase 5.5 (확장)** | 6~12개월 | **1,500** | 10% | `/employers` 양면 시장 점화, 공식 채용 딥링크 20건 이상 |
| **Phase 6 (커뮤니티)** | 12~24개월 | **3,500** | 22% | 익명 라운지 + 경력 타임라인 도입으로 재방문 2배 |
| **Phase 7 (앱 + "아름" 확장)** | 24~36개월 | **8,000** | 51% | 모바일 앱 전환 + 항공사 화물 부서 + 포워더 종합까지 흡수 |

### Supporting SOM 지표 (관리자 `/admin/dashboard` 추적)
| 지표 | Phase 5 목표 | 근거 |
|---|---|---|
| 이메일 구독자 누적 | 1,500 | Open rate 40% → WAU 500 역산 (Morning Brew PMF 기준) |
| 4주 후 유지율 | 40% 이상 | Reforge PMF 기준 (reforge.com) |
| 주간 신규 승인 채용 공고 | 15건 이상 | Day 1 시드 20건 + 공식 딥링크 주간 자동 업데이트 |
| MUV (월간 고유 방문자) | 2,000 | WAU × 4 배수 가정 |
| 유입 경로 다양성 | 3개 이상 채널 | organic / referral / direct 균형 |

### SOM이 의미하는 것
- **이 숫자가 바로 아름 카고의 "가설 검증 기준"**.
- Phase 5 3개월 시점에 WAU **150명** 미만이면 → Growth Loop 가설 재검토 (카페/LinkedIn 채널 이외 추가 유입 경로 탐색).
- Phase 5 6개월 시점에 WAU **500명** 미달이면 → C1 핀포인트 재검증 (OQ-R13 C1 5명 인터뷰 재실행).
- Phase 5.5 12개월 시점에 `/employers` 문의 **5건** 미달이면 → B1 Pain 재검증 또는 양면 시장 전략 재설계.

## 4. Segment Map — 2×2 매트릭스

### 축 정의
- **X축 (수평)**: 도메인 전문성 (낮음 ← → 높음)
- **Y축 (수직)**: 디지털 능동성 / 외부 정보 소비 빈도 (낮음 ↓ ↑ 높음)

```
  디지털 능동성 (높음)
       ↑
       │
       │   ┌─────────────────────┬─────────────────────┐
       │   │ Q2: 저전문·고능동    │ Q1: 고전문·고능동    │
       │   │                    │ ★ 핵심 SOM 타겟 ★    │
       │   │ • A1 학과 재학생     │ • C1 2~5년차 영업    │
       │   │ • C2 1년차 신입      │ • C3 6~10년차 시니어 │
       │   │                    │ • B1 채용 담당자     │
       │   │                    │                     │
       │   ├─────────────────────┼─────────────────────┤
       │   │ Q3: 저전문·저능동    │ Q4: 고전문·저능동    │
       │   │                    │                     │
       │   │ • 학부모·취업 관망층 │ • 11년+ 시니어       │
       │   │ • N1 항공 관심 일반 │ • 오프라인 네트워크  │
       │   │                    │   기반 이직 완료자   │
       │   └─────────────────────┴─────────────────────┘
       │                                         →
       └──────────────────────────────  도메인 전문성 (높음)
```

### 해석
- **Q1 (★ 핵심 SOM — Sweet Spot)**: 도메인 전문성과 디지털 능동성 모두 높은 층.
  - **C1 2~5년차 영업·오퍼**: 매일 아침 업계 뉴스 확인 + 이직 욕구 + 외부 정보 갈증이 겹치는 핀포인트.
  - **C3 6~10년차 시니어**: 팀 운영·업계 흐름 모니터링. 에디터 Pick 품질 평가자.
  - **B1 채용 담당자**: 채용난 해소 욕구. Phase 5.5 `/employers` 타겟.
- **Q2 (저전문·고능동)**: 디지털에 능숙하지만 도메인 지식 축적 중. 뉴스 다이제스트가 **학습 도구**로 기능.
  - **A1 학과 재학생**: 취업 준비 단계. Phase 5.5 `/jobs` 신입 공고 타겟.
  - **C2 1년차 신입**: 입사 직후 업계 흐름 파악 필요.
- **Q3**: 우선순위 낮음. MVP에서는 배제. (N1 항공 관심 일반인 = 마니아층은 Phase 7 확장 시 재고려)
- **Q4 (고전문·저능동)**: 11년+ 시니어는 오프라인 네트워크만으로 업무·이직 해결 → 아름 카고가 제공할 가치가 제한적. Phase 7에서 "아름" 커뮤니티로 일부 재공략.

### MVP 핵심 타겟 우선순위
1. **🥇 C1 2~5년차 콘솔사·포워더 영업·오퍼** — 약 4,000명. 핀포인트. WAU 중추.
2. **🥈 C3 6~10년차 시니어** — 약 2,450명. 재방문 + 브랜드 앰배서더 가능성.
3. **🥉 B1 채용 담당자** — 약 255명. Phase 5.5 양면 시장 점화.
4. **A1 학과 재학·졸업예정자** — 약 2,375명. 장기 깔때기 상단.
5. **C2 1년차 신입** — 약 1,350명. 뉴스 다이제스트 학습 채널.

## 5. 경쟁 공백 — P05 "현직자 시선" (Moat 핵심)

아름 카고의 핵심 차별화는 **"11년차 현직자가 직접 큐레이션한 업계 뉴스 + 채용"** — 이는 현재 한국 항공 화물 시장에서 **100% 공백 영역**이다.

### 기존 대체재 vs 아름 카고

| 영역 | 기존 대체재 | 한계 | 아름 카고 차별화 |
|---|---|---|---|
| **뉴스** | 네이버 뉴스 "항공화물" 키워드 | 일반 기자 작성, 맥락 없음 | 11년차 현직자 에디터 Pick + 친근 톤 |
| **해외 뉴스** | Loadstar, Air Cargo News (영문) | 언어 장벽 + 유료 | GPT-4o-mini 번역 + 핵심 요약 무료 |
| **채용** | 사람인 / 워크넷 / 지인 추천 | 키워드 난해 + 파편화 | 화물 특화 키워드 + 큐레이션 + 기업 딥링크 |
| **커뮤니티** | 네이버 카페, 오픈카톡방, LinkedIn | 분산·폐쇄 | Phase 6 익명 라운지 + 경력 타임라인 |
| **운임·capacity** | 현장 Excel 공유 | 실시간 부재 | Phase 5.5 기종 capacity + Phase 6 TAC Index 참조 |

### SOM 경쟁 비중 가정
- 현재 어떤 플랫폼도 이 공백을 점유하지 않음 → Phase 5에서 **P01(뉴스) 점유율 20%, P03(채용) 점유율 10%** 시드만 확보해도 WAU 500 달성 가능.
- P05 "현직자 시선" 영역은 **사용자 본인 11년차 경험치가 유일한 Moat**이므로 경쟁 진입 장벽이 매우 높음.

## 6. TAM/SAM/SOM의 PRD 반영
